Important HTML tags

This page is designed as a quick reference for some commont tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, just ones you'll see and use often.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table of data
table row table row include a row of data
table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>techinical sophistication</em> technical sophistication
strong strong, or bold make text strong, or bold <strong>Seattle Mariners</strong> Seattle Mariners
a reference create a hyperlink, or "anchor" <a href="http://www.nytimes.com"/> www.nytimes.com
img image insert an image <img src="images/kitten.jpg">
